# Python图书借阅管理系统简介

## 核心功能
- 登录注册
- 图书借阅
- 归还图书
- 图书信息管理
- 用户管理
- 借阅统计
- 图书统计

## 适用场景
- 图书馆管理
- 学校图书馆
- 公共图书馆
- 企业图书馆

- 使用Python Flask框架
- 采用HTML、CSS、JavaScript、jQuery进行前端开发
- 后端技术包括Python和MySQL
- 支持Python 3.0以上版本和MySQL 8.0
